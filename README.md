# Server Utils (RU/EN)

[Русский](#русский) | [English](#english)

---

## Русский

Сборник полезных серверных фич: создание невидимых рамок, невидимые светящиеся блоки, отображение ролей игроков в статус-баре при наведении и кастомный дизайн списка Tab с заголовками и подвалами.

### Фичи
- **Невидимые рамки**: Клик взрывным зельем невидимости по рамке с предметом делает её невидимой.
- **Индикатор ролей**: Отображает кастомные роли игроков в статус-баре при наведении курсора.

### Настройка конфигурации (config.yml)
```yaml
lang: "ru"

messages:
  ru:
    invisible_frame_name: "Невидимая рамка"
    tab_title: "My Server"
    tab_online: "Онлайн: "
    tab_role: "Ваша роль: "
    tab_ping: "Пинг: "
    tab_ping_ms: " мс"
```

## English

A clean and configurable server utility plugin for Bukkit/Paper servers. It adds several visual and utility adjustments to enhance vanilla gameplay.

### Features
- **Invisible Item Frames**: Splash an invisibility potion on a frame to turn it invisible.
- **Role Display**: Shows player roles in the action bar when pointing at them.

### Configuration Example (config.yml)
```yaml
lang: "ru"

messages:
  ru:
    invisible_frame_name: "Невидимая рамка"
    tab_title: "My Server"
    tab_online: "Онлайн: "
    tab_role: "Ваша роль: "
    tab_ping: "Пинг: "
    tab_ping_ms: " мс"
```
